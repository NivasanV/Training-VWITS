---- Component ----

    - Component is independent reusable code
    - It is a JavaScript file which returns JSX
    - I React, Component can be of following 2 types:
        1. Function Component:
            -It is new popular approach of writting React application
            -With the introduction of hooks in React, Function based Components
             heavily used
        2. Class Component:
            -Class based Component can still be used but it is old approach
             of writting React application
            -This style was used prior to introduction of hooks

-------- JSX --------
    - JavaScript XML
    - It allows to write html in React component.
    - It makes easy to write html in React.
