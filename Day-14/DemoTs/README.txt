------- Type Script -------
    Also known as TS in short
    TypeScript is superset of javascript which adds typing to the language.
    Browser understant javascript ans not TypeScript
    TypeScript file is converted into javascript then given back to Browser
    the process of converting TypeScript to javascript is known is confilation.
    to do this we need TypeScript compiler which we need to first install
    It is done as follow:
        1.to install    : npm install -g TypeScript
        2.to check      : tcs -v 
        3.to run        : tsc filename.ts

Module
    Module helps you to  break  our code into seperate file
    Each file is responsible for specific task
    each file is known as Module
    In order to use code outside the Module you must export it
    outside the Module you must import it
