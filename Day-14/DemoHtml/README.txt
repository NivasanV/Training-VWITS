-----HTML-----
HTML stands for Hyper Text Markup Language
HTML is used to decide the structure of the web page
like XML it is markup language hence we write tags while writing html
HTML has so many built-in tags
meaning of the tag is already defined and browser knows how to render a specific tag


------CSS------
stands for Cascading style Sheet
it is used to decide look and feel of the web page
eg: color combination, font size, font style 
CSS consist of built-in properties where each property is key:value pair
it is written as key:value
