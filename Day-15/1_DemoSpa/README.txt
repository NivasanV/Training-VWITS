SPA:
    -Stands for Single Page Application
    -Angular is a framwork to built client-side application to Single Page Application

    -SPA is a new style of web application development in which portion of the current page is rewritten
    dinamicaly by fetching data from the server instead of reloading the entire new page.
    -Benefit of SPA is it gives better user experience
    -It needs less bandwidth

    -SPA application can be build using java script and ajex but not only becomes diffcult build
    error prone diffcult to manage
    -Hence, lot of client side framwork and libraries are development in recent past
    -To name few Angular, React, Veujs and so on
