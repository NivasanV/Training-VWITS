In real-world applications are divided into client and server.
Server accepts the request from client and processes it and send the response back.
It is server must run first and must keep running always.
Client initiate the communication b/w server and client.
Client must be aware of IP address where server is running and the PORT number.
Client must be aware of IP address and PORT number for communicating with the sever.
A socket is a combination of IP and PORT.
		(Socket = IP + PORT)
