JSON(JavaScript Object Notation)
JavaScript object is written in {}
JavaScript object is written as key:value pair
keys are in ""
if values are of type string, then written in ""
Multiple key:value pairs are separated by ','
JSON Array is surrounded  with []

JSON is the most common and popular format for exchanging the data among applications
JSON has gained the popularity  in recent year with the rising popularity of client-side JS frameworks such as React, Angular
JSON is technology independent way of exchanging the data.
